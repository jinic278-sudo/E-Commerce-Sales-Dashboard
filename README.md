# E-Commerce Sales Dashboard

## Project Overview
This project is an interactive E-Commerce Sales Dashboard built in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, and KPI Cards. The dashboard provides insights into sales performance, profit, customer segments, regional sales, and monthly sales trends.

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Average Discount KPI
- Sales by Region
- Profit by Category
- Sales by Segment
- Monthly Sales Trend
- Top 10 States by Sales
- Interactive Slicers
  - Ship Mode
  - Region
  - Category
  - Segment

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Design

## Key Insights
- California recorded the highest sales among all states.
- Technology generated the highest profit.
- Consumer segment contributed the largest share of sales.
- Sales showed an overall upward trend over time.
